**问题描述：**
给定一个链表，将链表右移 k 个位置。

**解题思路：**
- 使用快慢指针找到链表的分割点
- 快指针先走 k 步，然后快慢指针同时前进
- 调整指针完成旋转

**复杂度分析：**
- 时间复杂度：O(n)
- 空间复杂度：O(1)
