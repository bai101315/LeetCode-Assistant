## 题目描述

给定一个 `n x n` 的二维矩阵 `matrix`，将图像顺时针旋转 **90** 度。

要求 **原地** 旋转图像，即直接修改输入的二维矩阵，**不得** 分配另一个二维矩阵来进行旋转。

---

## 示例

### 示例 1:
```
输入: matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出: [[7,4,1],[8,5,2],[9,6,3]]
```

### 示例 2:
```
输入: matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
输出: [[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]
```

---

## 约束条件

- `n == matrix.length == matrix[i].length`
- `1 <= n <= 20`
- `-1000 <= matrix[i][j] <= 1000`

## 解题思路

### 方法一: 逐层旋转（四指针法）

从外层到内层，对每一层进行四个边界元素的循环交换：

1. **转置 + 翻转列**: 先将矩阵转置，再翻转每一行
2. **分层处理**: 从外层向内层，按顺时针顺序交换四个对应位置的元素

### 核心步骤:
1. 对矩阵进行转置（沿主对角线交换）
2. 翻转每一行（左右交换）
3. 或者使用四元素交换：top → right → bottom → left → top

**时间复杂度**: O(n²)  
**空间复杂度**: O(1)
